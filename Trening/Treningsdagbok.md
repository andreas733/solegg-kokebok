# Treningsdagbok 🏃

> Løpedagbok for sommeren 2026 og fremover.
> Mål: bygge kondisjon med 3–4 økter per uke.
> Utgangspunkt: ~60 min på 10 km (6:00/km)

---

## Format per økt
```
**Dato:** YYYY-MM-DD
**Type:** Rolig / Intervaller / Langtur
**Distanse:** X km
**Tid:** HH:MM
**Snittfart:** X:XX/km
**Kommentar:** ...
```
